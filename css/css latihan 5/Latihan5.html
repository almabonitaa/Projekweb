Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah web yang memiliki fungsimengatur tampilan dokumen web lain, yaitu HTML. Bebebrapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf, jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadaanya tidak menggantikan HTML melainkan sebagai pelengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. Berikut ini adalah contoh dokumen CSS.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline Style : Menuliskan kode CSS secara langsung pada tag yang akan diubah atau diatur style-nya
  2. Internal/Embedded Style : menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/linked Styles : Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
